# Cabinet Avocat Gurev Dorina - Landing Page

A professional landing page for "Cabinet Avocat Gurev Dorina," a law office based in Chișinău, Moldova. The design features a refined burgundy/brown aesthetic with elegant typography, smooth animations, and a trustworthy professional appearance.

## Live Demo

**[View Live Site](https://m3ower.github.io/pw-lab2/)**

## Description

This is a single-page website for Cabinet Avocat Gurev Dorina, featuring:
- Attorney credentials and professional information
- Legal service offerings across multiple practice areas
- Contact form and office information
- Responsive design for all devices
- Burgundy/brown professional color scheme

**Technologies:** HTML5, CSS3 (no JavaScript)
